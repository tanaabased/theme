# Theme

## Development

Start by [installing bun](https://bun.sh/).

Clone and install deps:

```sh
git clone https://github.com/tanaabased/theme.git
cd theme
bun install
```

Run dev server

```sh
bun run dev
```

Build and preview for production:

```sh
bun run build
bun run preview
```

Lint:

```sh
bun run lint
```
